Cluster analysis or clustering is the task of grouping a set of objects in such a way that objects in the same group (called a cluster) are more similar (in some sense) 
to each other than to those in other groups (clusters). It is a main task of exploratory data mining, and a common technique for statistical data analysis, used in many fields, 
including machine learning, pattern recognition, image analysis, information retrieval, bioinformatics, data compression, and computer graphics <a target="_blank" href="https://en.wikipedia.org/wiki/Cluster_analysis">[more]</a>.
